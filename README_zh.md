# PromptTemplates-LLM

一个我自己整理和搭出来的 AI 提示词工具仓库。

这个项目现在的定位不是原始的 LangGPT 展示仓库，而是我自己日常使用的提示词工作台和模板库。它把本地网页工具、常用提示词模板、操作说明、解释文档、示例和参考资料放在了一起，方便我平时直接拿来用，也方便后续继续扩展。

[English](README.md) | [简体中文](README_zh.md) | [日本語](README_ja.md)

## 这里面有什么

- `PromptWorkbench.html`
  本地提示词工作台，打开就能用。
- `Prompts/`
  我自己常用的提示词模板目录。
- `操作手册.md`
  讲怎么启动、怎么打开、怎么使用这个工作台。
- `解释手册.md`
  讲这个项目为什么这么设计、每个部分负责什么。
- `Docs/`、`Papers/`、`examples/`
  作为提示词设计和学习过程里的补充资料继续保留。

## 当前功能

- 本地网页工作台，运行地址是 `http://localhost:18888/`
- 覆盖聊天、学习、写作、工作、规划、比较、代码、生活场景的模板
- 支持模板搜索、收藏、本地保存和结构化提示词生成
- 当前版本足够轻，不需要前端打包流程就能直接使用

## 快速开始

### 1. 启动本地服务

Windows 下可以直接运行：

```bat
run-server.cmd
```

也可以手动执行：

```powershell
node server.js
```

### 2. 打开页面

```text
http://localhost:18888/
```

如果当前环境里 `localhost` 不稳定，也可以用：

```text
http://127.0.0.1:18888/
```

## 我现在的使用方式

1. 打开工作台
2. 选一个场景模板
3. 填主题、背景或原文
4. 调整语气、长度、输出方式
5. 复制生成结果，发给我常用的 AI
