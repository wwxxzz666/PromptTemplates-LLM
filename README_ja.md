# PromptTemplates-LLM

日常的に AI を使うための、個人向けプロンプトテンプレート集とローカル作業環境です。

このリポジトリは、もともとの LangGPT の見せ方ではなく、私自身の実用プロジェクトとして整理し直したものです。ローカルで使える Prompt Workbench、再利用しやすいテンプレート、使い方ガイド、補足資料をひとつにまとめています。

[English](README.md) | [简体中文](README_zh.md) | [日本語](README_ja.md)

## 内容

- `PromptWorkbench.html`
  すぐに使えるローカルのプロンプト作業画面
- `Prompts/`
  よく使うプロンプトテンプレート集
- `操作手册.md`
  起動方法と使い方のガイド
- `解释手册.md`
  設計意図と構成の説明
- `Docs/`、`Papers/`、`examples/`
  学習用メモ、参考資料、サンプル

## 主な機能

- `http://localhost:18888/` で動くローカル Web ページ
- 会話、学習、文章作成、仕事、計画、比較、コード作業向けテンプレート
- 検索、お気に入り、ローカル保存、構造化プロンプト生成
- 現状はビルド不要の軽量構成

## クイックスタート

### 1. ローカルサーバーを起動

Windows:

```bat
run-server.cmd
```

または:

```powershell
node server.js
```

### 2. アプリを開く

```text
http://localhost:18888/
```

必要ならこちらでも開けます:

```text
http://127.0.0.1:18888/
```
