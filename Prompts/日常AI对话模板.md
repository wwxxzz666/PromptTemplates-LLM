# 日常 AI 对话提示词模板

这份模板集是给“平常和 AI 聊天、问问题、做事情”直接用的。  
你可以把 `<...>` 里的内容替换成自己的需求，或者整段复制给 AI。

## 1. 通用日常助手

```markdown
你是我的日常 AI 助手。请先判断我这次想要的是解释、写作、总结、规划、对比、翻译、代码、搜索还是复盘。
如果信息不够，请先问我最多 3 个关键问题。
如果信息够了，请直接给我结果，并尽量用清单、步骤、表格或对比的方式输出。
请优先给出最实用的答案，不要空泛。
```

## 2. 解释概念

```markdown
请用“我能听懂”的方式解释 <概念>。
要求：
1. 先给一句话结论
2. 再用生活例子解释
3. 再讲容易混淆的点
4. 最后给我一个 3 行以内的记忆版总结
如果有专业术语，请顺手翻成大白话。
```

## 3. 学习辅导

```markdown
我在学 <主题>，请你当我的耐心老师。
请按这个顺序帮我：
1. 先判断我现在大概处在什么水平
2. 再用循序渐进的方式讲清楚
3. 每讲完一个小点，给我一个小练习
4. 最后出 3 道由易到难的巩固题
如果我回答错了，请直接指出问题并帮我纠正。
```

## 4. 写作润色

```markdown
请帮我润色下面这段文字：
<原文>

目标：
1. 保留我的原意
2. 让表达更自然、更顺
3. 去掉重复和废话
4. 按我想要的风格调整：<口语/正式/简洁/有感染力/适合发朋友圈>
如果你觉得有更好的结构，也可以顺手帮我重排。
```

## 5. 总结提炼

```markdown
请帮我总结下面这段内容：
<内容>

请输出这 4 部分：
1. 核心结论
2. 关键要点
3. 容易忽略的细节
4. 如果我要行动，下一步该做什么
如果内容太长，请优先提炼出最重要的 20%。
```

## 6. 头脑风暴

```markdown
我想围绕 <主题> 做头脑风暴。
请不要急着下结论，先给我 10 个不同方向的想法。
要求：
1. 尽量覆盖不同角度
2. 不要只给常规答案
3. 每个想法用 1 句话说明
4. 最后帮我挑出最值得继续展开的 3 个
```

## 7. 决策比较

```markdown
请帮我比较下面这些选项：
<选项A>
<选项B>
<选项C>

请按以下维度分析：
1. 适合什么情况
2. 优点
3. 缺点
4. 成本/收益
5. 我这种情况更适合哪个
最后请直接给出你的建议，不要只列信息。
```

## 8. 计划安排

```markdown
我想完成 <目标>，请你帮我做一个现实可执行的计划。
请按下面格式输出：
1. 目标拆解
2. 每一步要做什么
3. 时间安排
4. 可能遇到的阻碍
5. 备用方案
要求计划要具体，不要空喊口号。
```

## 9. 会议纪要 / 聊天整理

```markdown
请把下面这段聊天记录/会议记录整理成可执行纪要：
<内容>

请输出：
1. 主题
2. 主要结论
3. 待办事项
4. 负责人
5. 截止时间
6. 需要继续确认的问题
如果信息不完整，请帮我标出来。
```

## 10. 代码助手

```markdown
请你当我的代码助手，帮助我处理 <语言/项目> 里的问题。
请先判断这是：
1. 排查 bug
2. 重构
3. 优化性能
4. 补功能
5. 解释代码

然后直接给我：
1. 问题原因
2. 修改建议
3. 参考代码
4. 需要注意的副作用
如果你需要更多上下文，先问我最关键的问题。
```

## 11. 翻译 / 双语改写

```markdown
请把下面这段内容翻译成 <目标语言>：
<原文>

要求：
1. 保留原意
2. 让语气自然
3. 如果是口语，翻得像真人会说的话
4. 如果是正式内容，保持专业
如果有歧义，请告诉我你是怎么处理的。
```

## 12. 旅行规划

```markdown
请帮我规划一次 <城市/地区> 的 <天数> 天游玩。
我的预算是 <预算>，偏好是 <美食/博物馆/自然/轻松/亲子/拍照>。
请给我：
1. 每天行程
2. 交通建议
3. 餐饮建议
4. 预算分配
5. 备选方案
6. 行前提醒
请尽量做得省心一点，不要排得太赶。
```

## 13. 求职 / 简历

```markdown
请你当我的求职助手，帮我优化 <简历/自我介绍/面试回答/求职信>。
我的背景是：
<背景信息>

请输出：
1. 可以直接使用的版本
2. 哪些地方需要改得更有竞争力
3. 哪些表达太空泛，需要换成结果导向的说法
4. 如果是面试题，请给我更自然的回答
```

## 14. 情绪支持 / 陪我聊聊

```markdown
我现在有点 <焦虑/低落/烦躁/迷茫>，先不要急着给建议。
请先认真听我说，帮我把感受整理清楚。
然后再给我：
1. 你理解到的我的状态
2. 可能的原因
3. 我现在能做的 1 到 3 个小步骤
4. 一句不空洞、能让我稳一点的话
请温和一点，但也真实一点。
```

## 15. 日常购物 / 对比选择

```markdown
我想买 <物品>，预算是 <预算>，需求是 <需求>。
请帮我比较不同选择，并告诉我：
1. 适合我的关键标准
2. 推荐的选购思路
3. 需要避开的坑
4. 如果预算更高/更低，怎么选更划算
最后请直接给我一个清晰建议。
```

## 16. 万能追问模板

```markdown
请继续往下帮我想，不要停在表面。
如果你觉得信息不够，就先问我最关键的 1 到 3 个问题。
如果你能直接推进，请直接给我下一步最有价值的内容。
我更想要能落地的答案，不想只听概念。
```

## 使用建议

- 如果你只是想日常聊天，优先用「通用日常助手」。
- 如果你是明确任务，直接用对应场景模板。
- 如果你想让 AI 更像你的个人搭子，可以在每个模板最后加一句：
  - “语气更轻松一点”
  - “回答更短一点”
  - “先给结论，再给理由”
  - “如果有风险，请先提醒我”
